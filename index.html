    About Me

    Hey there, I'm Quinn. I'm a third year student at Oregon State University studying Electrical and Computer Engineering. My background is in embedded microcontroller programming, but I'm also looking into analog circuitry design and radio frequency applications as well. I like simplicity. I enjoy learning about complex systems by building my own, stripped to the minimum required to work, then adding functionality only as needed. I'm a general-class amateur radio operator under the call sign KK7NQV.

    Highly Recommended Online Courses

    • Nand2Tetris — Design a computer from scrath in HDL, write assembly programs, create compilers, develop an operating system, and build a graphical application. I completed this course and it really helped with my conceptual understanding of how all parts of a computer work together by implementing all levels of abstraction.
    • Ben Eater: Build a 6502 Computer — Learn how computers work by building and programming a computer with the clasic 6502 microprocessor. This course really helped me get comfortable writing low-level code to control hardware. I also learned lots of valuable skills debugging hardware.
